Acerca de la Aplicación

Este módulo permite comunicarse en tiempo real a través de una interfaz amigable. Las características incluyen:

  • Mensajería en tiempo real
  • Estado de presencia de los usuarios (Online/Offline)
  • Confirmaciones de entrega y lectura de mensajes
  • Funcionalidad de búsqueda para mensajes

Tecnologías Utilizadas

La aplicación está construida utilizando las siguientes tecnologías:

  • Django para el backend
  • Autenticacion con Jason Web Token
  • Bootstrap para diseño responsivo
  • Base de Datos PostgreSQL

Cómo Usar

Para comenzar:

  1. Clona el repositorio desde GitHub.
  2. Agrega el entorno virtual python -m venv venv && source venv/bin/activate.
  3. Instala las dependencias requeridas usando pip install -r requirements.txt.
  4. Agregar las migraciones python manage.py makemigrations && python manage.py migrate.
  5. Ejecuta la aplicación usando python manage.py runserver.
  6. Accede a la aplicación en tu navegador web en http://127.0.0.1:8000.
